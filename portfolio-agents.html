Portfolio Intelligence

A read-only analysis tool that monitors markets, evaluates your holdings, and delivers insights you can act on

Four AI agents work together to scan macroeconomic news, track security-specific events, analyze portfolio performance and risk, then synthesize everything into actionable trade readouts and podcast-style audio briefings — all in a native Mac app.

Agent Architecture

Four agents, one cohesive picture

Each agent has a dedicated role, data sources, and output format. The orchestrator coordinates their execution order and passes context between them so the Synthesizer can cross-reference everything into actionable intelligence.

🌍
Macro Scanner
Agent 1 · World & US News

Scans broad news sources for macroeconomic events — Fed policy, GDP, employment, geopolitical shifts, trade policy — and classifies each by potential market impact and affected sectors.

NewsAPI Finnhub News Impact Ratings Sector Tags
🔍
Security Scanner
Agent 2 · Holdings-Specific Intelligence

Monitors news, earnings, analyst ratings, and corporate actions for each holding in the portfolio. Maintains a forward-looking events calendar and evaluates custom alert rules.

Finnhub Company News Earnings Calendar Custom Alerts Sentiment
📈
Portfolio Analyst
Agent 3 · Performance, Risk & Tax

Evaluates holdings from Fidelity — P&L, allocation drift, benchmark comparison, scenario stress testing (2008, COVID, dot-com), and tax-loss harvesting opportunities. Tracks progress toward financial goals.

Fidelity CSV / API Finnhub Prices Drift Alerts Stress Tests
🧠
Synthesizer + Audio
Agent 4 · Cross-Reference & Recommendations

Combines all agent outputs into a cohesive briefing. Generates per-holding "Stock Story" narratives, thematic exposure analysis, trade readouts, and podcast-style audio summaries via ElevenLabs.

Trade Readouts Stock Stories Audio Briefing Theme Analysis
Feature Set

Inspired by the best platforms

Features drawn from Merrill Lynch, Morgan Stanley, Fidelity, Schwab/thinkorswim, and LPL Financial — rebuilt as a personal, read-only analysis tool.

🗺️
Portfolio Heatmap

Treemap visualization sized by allocation, colored by performance. One glance shows where the portfolio is thriving or bleeding.

Inspired by Merrill Lynch, thinkorswim

💥
Scenario Stress Testing

Replay historical crises (2008, COVID, dot-com) against current holdings to estimate portfolio vulnerability under extreme conditions.

Inspired by Merrill Lynch, Morgan Stanley Aladdin

⚖️
Drift Detection

Define target allocations, get alerted when market movements push the portfolio off-target, with specific rebalancing suggestions.

Inspired by LPL ClientWorks, Schwab Intelligent Portfolios

🧾
Tax-Loss Harvesting

Scan unrealized losses against realized gains and suggest tax-aware trades. Especially valuable near year-end.

Inspired by Morgan Stanley Parametric, Fidelity

📖
Stock Stories

AI-generated per-holding narratives covering performance, catalysts, sentiment, and portfolio context — not just raw data tables.

Inspired by Merrill Lynch Stock Story & Fund Story

🔮
What-If Simulator

Model adding or removing positions without executing trades. See how hypothetical changes affect allocation, risk, and goals.

Inspired by thinkorswim paperMoney

🎯
Goal Tracking

Set financial targets with projected trajectories based on current growth rate. Visual progress bars show if you're ahead or behind schedule.

Inspired by Fidelity Planning Center, Morgan Stanley GPS

🎙️
Podcast Briefings

Weekly audio summaries generated by ElevenLabs with two AI host voices. Listen to your portfolio analysis like a podcast while commuting.

Inspired by Google NotebookLM Audio Overviews

📋
Trade Readouts

Clear, actionable trade recommendations — ticker, action, quantity, rationale — formatted for quick manual entry in Fidelity. You decide what to act on.

Tech Stack

Built with Python, designed for Mac

🤖
CrewAI → LangGraph
Agent Orchestration

CrewAI for role-based multi-agent coordination. Migration path to LangGraph for persistent state and decision history tracking.

🧠
Claude API
LLM Reasoning Engine

Anthropic's Claude powers each agent's analysis, classification, and narrative generation via the Python SDK.

📊
Dash + Plotly
Dashboard Framework

Real-time web dashboard with callbacks, native financial chart types, and production-grade performance.

📉
Lightweight Charts
Financial Charting (TradingView)

Open-source, 45KB, candlestick charts with volume, technical indicators, zoom, pan, and crosshairs.

🖥️
Pywebview
Native Mac Window

Wraps the Dash app in a native macOS Cocoa window with dock icon, system notifications, and Python-to-JS bridge.

🎙️
ElevenLabs
Podcast Audio Generation

Two-voice text-to-speech API that converts analysis scripts into podcast-style audio briefings.
